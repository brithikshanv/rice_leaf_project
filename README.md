## Problem Statement
Build a machine learning model to classify major rice leaf diseases using image data.

## Tasks
- **Task 1:** Perform exploratory data analysis (EDA) on the dataset

- **Task 3:** Analyze techniques such as data augmentation and compare model performance  

## Dataset
- **Total Images:** 120 JPG images  
- **Classes (40 images each):**
  - Leaf smut
  - Brown spot
